# magnetic-soft-robotic-gripper
Design, fabrication, and simulation of a bio-inspired magnetic soft robotic gripper using Ecoflex 00-50 and Magnetite.

Overview
This project presents the design, simulation, and fabrication of a magnetically actuated soft robotic gripper capable of handling delicate objects.

The gripper is developed using a silicone (Ecoflex) and magnetite composite, enabling deformation under external magnetic fields.
Objectives
Design soft robotic gripper geometries using CAD
Perform FEA & Magnetic FEM simulations (ANSYS)
Develop magneto-active composite material
Fabricate prototype using 3D printing
Validate real-world performance
